Overview

Ut Video Codec Suite is a lossless video codec for Windows, which is implemented as a VCM codec (also called "VfW codec").

Environment

Minimum requirement

OS
Windows XP or later
CPU
i686-compatible CPU with SSE2 support
(e.g. Pentium 4 or later)

Microsoft Visual C++ 2005 SP1 Redistributable Package (x86) is also required. Installation of Ut Video Codec Suite will fail unless Redistributable Package is installed.
